Les commandes indispensables de git:

  1. git init: pour initianiliser un nouveau repository
  2. git log : affiche l'historique de validation des commits
  3. git add . : afin que les modifications soient prisent en compte
  4. git commit -m "mentionner les mofifications apportées": pour enregistrer en local votre travail, il n'écrase pas les sauvegardes précédentes
  5. git push origin nomDeLaBranch : récupère les précédents commits pour les mettre en ligne, ils seront alors accessibles depuis GitHub
  6. git remote add origin https://github.com/SoniaB78/monRepository : Lorsque le fichier à déjà été créé en local et qu'on à créé un repo sur GitHub pour le stocker (ne rien mettre dedans avant de taper cette ligne)
