Les commandes indispensables de git:

  1. git init: pour initianiliser un nouveau repository
  2. git log : pour s'identifier auprés de github
  3. git add . : afin que les modifications soient prisent en comptes
  4. git commit -m "mentionner les modifications apportées": pour enregistrer en local votre travail,il n'ecrase pas les sauvegardes précédentes
  5. git push origin NomDeLaBranch
